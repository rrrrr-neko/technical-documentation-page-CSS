Краткая сводка о CSS

What is CSS

CSS (Cascading Style Sheets) — это язык таблиц стилей, который используется для описания внешнего вида веб-страниц. Он позволяет определять стили элементов HTML, таких как шрифты, цвета, отступы и т. д.

        
      .example {
        color: red; /* Устанавливаем цвет текста */
      }

История CSS началась в 1994 году с разработки языка Cascading HTML Style Sheets в компании Netscape Communications. В 1996 году была создана первая версия стандарта CSS, которая получила название CSS1. С тех пор стандарт постоянно развивался и дополнялся новыми возможностями. В настоящее время актуальной версией является CSS3.

Selectors

Селекторы в CSS — это правила, которые применяются к определённым элементам веб-страницы. Они используются для стилизации HTML-элементов.

    Существует несколько типов селекторов:
  • Универсальные селекторы (*) — применяются ко всем элементам на странице.
  • Селектор элемента (тег) — применяется к конкретному тегу, например, h1 или p.
  • ID-селектор (#id) — применяется к элементу с указанным идентификатором.
  • Класс-селектор. (.) — применяется к элементам с указанным классом.
  • Групповой селектор (элемент1 элемент2) — применяет стиль к нескольким элементам одновременно.
  • Дочерний селектор ($element1 > $element2$) — выбирает элементы, являющиеся непосредственными потомками указанного элемента.
  • Потомок селектор ($element1 $element2$) — выбирает все элементы, находящиеся внутри указанного элемента на любом уровне вложенности.
  • Псевдоклассы (:hover, :active, :focus) — добавляют стили при определённых состояниях элемента, таких как наведение курсора мыши, активация и фокусировка.
  • Псевдоэлементы (::before, ::after) — позволяют добавлять контент до или после содержимого элемента.

Margin Padding Border

Margin — это расстояние от внешнего края элемента до ближайшего соседнего элемента. Это невидимое пространство, которое используется для создания отступов между элементами на веб-странице. Пример: 


<!DOCTYPE html>
<html>
  <head>
    <style>
      div {
        width: 200px;
        height: 100px;
        background-color: yellow;
        margin: 50px; /* Устанавливаем внешний отступ */
      }
    </style>
  </head>
  <body>
    <h1>Пример использования margin</h1>
    <div></div>
  </body>
</html>
В этом примере у блока div установлен внешний отступ в 50 пикселей с помощью свойства margin.

Padding — это пространство внутри элемента, которое находится между его содержимым и границей. Padding также является невидимым пространством, но оно влияет на размер самого элемента. Пример: 


<!DOCTYPE html>
<html>
  <head>
    <style>
      div {
        padding: 30px; /* Устанавливаем внутренний отступ */
        border: 1px solid black;
        width: 400px;
        height: 200px;
      }
    </style>
  </head>
  <body>
    <h1>Пример использования padding</h1>
    <div>Текст внутри блока div</div>
  </body>
</html>
Здесь у блока div установлено внутреннее заполнение в 30 пикселей с помощью свойства padding.

Border — это линия, которая окружает элемент. Border может иметь различные стили (сплошная, пунктирная, двойная и т. д.), ширину и цвет. Пример: 

  
<!DOCTYPE html>
<html>
  <head>
    <style>
      div {
         border: 2px solid red; /* Устанавливаем границу */
        width: 400px;
        height: 200px;
      }
    </style>
  </head>
  <body>
    <h1>Пример использования border</h1>
    <div>Блок div с красной границей</div>
  </body>
</html>

Img CSS

Для подключения изображений в CSS используется свойство background-image. Оно позволяет установить фоновое изображение для элемента. Вот пример использования свойства background-image: 


<!DOCTYPE html>
<html>
  <head>
    <style>
      div {
        background-image: url("image.jpg"); /* Указываем путь к изображению */
        width: 400px;
        height: 200px;
      }
    </style>
  </head>
  <body>
    <h1>Пример использования background-image</h1>
  </body>
</html>

В этом примере у блока div установлено фоновое изображение с помощью свойства background-image. Путь к изображению указывается относительно файла со стилями (в данном случае это файл style.css).

Makets

Макеты в HTML и CSS — это способ организации элементов веб-страницы, таких как заголовки, абзацы, изображения и другие блоки контента. Макеты определяют структуру и внешний вид страницы, а также её адаптивность под разные размеры экрана.

Существует несколько основных принципов создания макетов:

  1. Блочная вёрстка. Это самый распространённый метод, при котором элементы располагаются в виде блоков (div). Блоки могут быть расположены друг под другом, рядом или вложенными друг в друга. Блочная вёрстка позволяет легко управлять размещением элементов на странице и их размерами.
  2. Флексбокс вёрстка (flexbox). Это более продвинутый метод, который использует свойства flex для управления размещением и размерами элементов. Флексбокс позволяет создавать гибкие макеты, которые адаптируются под разные размеры экрана.
  3. Грейдерная вёрстка (grid). Этот метод использует свойства grid для создания сетки из колонок и строк. Грейдерная вёрстка также позволяет создавать адаптивные макеты.
  4. Табличная вёрстка.** Этот метод используется для создания сложных макетов с таблицами и списками. Табличная вёрстка может быть сложной в управлении и не всегда подходит для адаптивных макетов.
  5. Смешанная вёрстка. В этом случае используются различные методы вёрстки в зависимости от требований к макету. Например, можно использовать блочную вёрстку для основного контента и флексбокс для боковых панелей.

Выбор метода вёрстки зависит от сложности макета, требований к адаптивности и других факторов.
